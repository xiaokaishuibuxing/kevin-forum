# Kevin论坛 · AI 产品经理社区官网

面向 AI 产品经理的企业宣传网站：整理最新行业规范与前沿技术，服务正在转型和成长中的 AI 产品人。

**在线预览**：`https://<你的用户名>.github.io/<仓库名>/`

## 项目简介

AI 产品经理是一个正在快速定义中的职业——规范在变、技术在变、企业的用人标准也在变。Kevin论坛把分散的信息整理成可执行的成长地图，并提供一套 AI 驱动的企业分析工具，帮助 AI PM 完成从学习到实战的闭环。

## 核心功能

### 1. AI 企业分析工具
输入任意企业名称，AI 按专业模版生成三类深度报告：
- **需求分析** — 业务背景、产品盘点、KANO 需求分层、竞品对标、P0/P1/P2 优先级建议
- **用户分析** — 用户细分、3 个完整用户画像、用户旅程地图、增长与留存策略
- **PRD 输出** — 可直接用于开发评审的完整 PRD（含 AI 能力设计、Eval 评估方案、兜底机制）

分析结果支持一键导出 **PPT（.pptx）** 与 **Excel（.xls）**。

### 2. 系统学习路径
三阶段学习地图（基础认知 → 核心技能 → 实战进阶），每阶段配套精选视频课程与文档资料链接。

### 3. 传统 PM vs AI PM 对比
五个关键维度（需求定义、评估方式、技能要求、交互设计、风险合规）的差异拆解，每项附转型注意事项。

### 4. 企业展示
论坛板块介绍、关于我们、数据成果、合作伙伴与社区之声、联系表单。

## 使用说明

- 网站为单文件静态页面（`index.html`），无需构建、无需后端
- AI 分析功能需要 [Anthropic API Key](https://console.anthropic.com/settings/keys)：在页面「AI 企业分析」区输入 `sk-ant-…` 开头的 Key 即可使用；Key 仅保存在访问者本地浏览器，不会上传到任何服务器
- API 调用按量计费，请确保账户有余额

## 部署

1. 新建 GitHub 仓库（Public）
2. 上传 `index.html` 到仓库根目录
3. Settings → Pages → Source 选择 `main` 分支 → Save
4. 约 1 分钟后网站上线

## 技术说明

- 纯前端单文件 HTML，内嵌全部样式与逻辑
- 字体：Noto Serif SC / Noto Sans SC / Space Grotesk（运行时从 Google Fonts 加载，离线自动降级为系统字体）
- PPT 导出基于 PptxGenJS（CDN 加载）
- AI 能力：Anthropic Messages API（claude-sonnet-4-5）
